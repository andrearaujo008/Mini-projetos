Atividade de miniprojetos em Python — Programação de Computadores (1º Semestre 2026)

Desafio 1 — Sistema de Votação
Registra votos para Ana, Bruno ou Carlos, valida entradas inválidas e exibe o resultado com vencedor ou empate.

Estruturas: lista, count(), while, condicionais
Executar: python desafio_01_votacao.py
Exemplo: entrada ana, bruno, ana, fim → saída O vencedor é: Ana


Desafio 2 — Editor com Desfazer
Simula um editor de texto onde o usuário digita palavras e pode desfazer a última (pilha LIFO).

Estruturas: lista como pilha, append(), pop()
Executar: python desafio_02_editor_pilha.py
Exemplo: adiciona minha e terra, desfaz → texto fica minha


Desafio 3 — Fila de Atendimento
Simula a fila da secretaria universitária, atendendo os alunos por ordem de chegada (fila FIFO).

Estruturas: lista como fila, append(), pop(0)
Executar: python desafio_03_fila_atendimento.py
Exemplo: Marco entra na fila → ao chamar próximo → Chamando aluno: Marco
